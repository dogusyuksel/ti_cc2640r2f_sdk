DVT Graph State/Event Categories


Some presentation types like State, Shape, Icon need color, shape or icon assignment based on the value of the field. The color/shape/icon settings can also be categorized so different channel use the same color scheme or use different ones.
Although a valid use case is that end user to come up with all the color/shape/icon and category assignments manually. This task however might easily get out of control because in most cases, we might not know what the possible categories and color assignment shall be especially if the data collection is handled with any data source that might not even exist today. So we've came up with a scheme that hopefully addresses this a combination of manual and autodiscover cases with some control over the color/shape/icons for a the known set of state values.
To specify the known set of state values, delete all categories other than Main category and specify visibility, color, shap, icon settings for each state/event. This will serve as the default setting for autodiscoverred categories.

Event/State Name


When autodiscovery/autoassign is enabled for categories, each channel that with "State", "Shape", or "Icon" presentation type, is listed as a category and the values within that channel, would be listed as valid states. All the possible states of the system also get listed under the Main category.

Visibility


Control whether or not show a state using the "Visible" checkbox.

Color


Configure what color to use for each state within a category

Shape


Configure what shape to use for each state, the shapes will be colored using the specified color for the state/event. Here are the supported shapes:
  • Circle
  • Line
    • - Line is different with other shapes in that a vertical line crossing all channel (which will be a two pixels thicker on the channel that is responsible for drawing), will be plotted. This comes handy for inserting markers that for example could mark start of frames.
  • Triangle
  • Square
  • Arrow
  • Alphabet
  • - Shows the first character of state inside a circle shaped border

Icon


The list of shapes that the graph supports is limited to the above list however user may easily add any icons they'd like to see by dropping the icon file to <install_dir>\dvt\icons and modifying state_icon.txt. Restarting the tool and assigning their icon name to the state.

Toolbar


Multi Select assignment and drilling down based on channel name works the same way as "Channels" tab.
