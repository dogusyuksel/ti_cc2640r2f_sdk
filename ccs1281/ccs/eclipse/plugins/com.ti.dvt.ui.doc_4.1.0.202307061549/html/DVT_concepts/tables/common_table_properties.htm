Table View Properties

You have options to control the column visibility, value display format (general, binary, decimal, hexadecimal, or scientific notation), text alignment, and font characteristics.

Set value format

How each value is displayed in a column of a table view is defined by the user with the Column Settings dialog. Clicking mouse button(MB3) in the table and selecting the Column Settings shortcut command displays this dialog.

  • Table column visibility (hide or display)

  • Value display format (general, binary, decimal, hexadecimal, or scientific notation). Some options permit you to control the display format (number of decimal places, hexadecimal format)

  • Text alignment (Left, Center, and Right)

  • Font characteristics of values (Font family, size, bold, italic, color, effect, and so forth)
