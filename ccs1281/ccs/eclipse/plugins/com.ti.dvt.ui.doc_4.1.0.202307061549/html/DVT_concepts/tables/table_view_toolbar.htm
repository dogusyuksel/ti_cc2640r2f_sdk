Table view toolbar

The toolbar controls table display options that apply to the entire table. Most of the functionality on a Table Toolbar is common to the functionality found on a Graph Toolbar. These table toolbar command icons display in each table view:

  • The (Groups) button associates or disassociates this view within a specified group. See the Groups Feature topic.

  • The (Refresh) button updates the view when pressed.

  • The (Find) button displays the Find dialog to search for a specific event or value. See the Find Feature topic.

  • The (Enable Filter) button displays the Filter dialog. See the Filter Feature topic for details.

  • The (Freeze Data Update) button to suspend the view's data refresh. Click again to resume the data refreshing at the defined interval.

  • (Minimize) icon reduces the view to just the dialog tabs and graph view toolbar.

  • (Maximize) icon enlarges the view to the entire available window size.

  • (Restore) icon returns a view to its original size, prior to using the Minimize or Maximize icons.

  • Use the (Auto Fit Columns) button to adjust each table column width so that all values are completely displayed.

  • Click the (Menu) icon and select Row Count to display an extra column showing the row count.
