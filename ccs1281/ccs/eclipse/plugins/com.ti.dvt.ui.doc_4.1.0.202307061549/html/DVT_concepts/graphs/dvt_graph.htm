DVT Graph

This kind of graph depicts state transitions, events and messages against time or some other sequence and historically replaces the original state and discrete graph. It can show groups of states/events/messages each forming a channel line with a common key value on the Y-axis. Data values that share that same are plotted on the same horizontal bar. Different data values are mainly distinguished by assignment of different colors and tooltips on mouse over reveal their value. Below is an example of such a graph


Presentation Types

Data values can be presented on the Y-Axis in any of the following forms:

  • State: Where data in the channel are drawn same way as our original "State Graph", basically each data value is assigned a color and a colored bar gets plotted using that color until a different state/data value is identified for the x-axis value.
  • Shape: It is good for event visualization, where data points in channels are drawn as Circle/Line/Triangle/Square/... color assignment logic is the same as State presentation.
  • Icon: Same as Shape this is good for event visualization, however unlike Shape presentation type, the data values don't dictate a color, they only dictate the icon to be used for plotting.
  • Message: It is good for presentation of long string messages where an "M" icon is drawn and text of message is shown as a tooltip on mouse-over. (default presentation type)
  • Value: Similar to VCD where for each value change, a new hexagonal is drawn and the value is presented inside the hexagonal.
  • Waveform: Plotted as a a 0/1 digital step waveform.
For State, Shape and Icon display formats, clicking on the channel name on Y-Axis, would expand the channel into possible states/events on separate lines, similar to to our original Discrete Graph


Tree Mode vs. Flat Mode

DVT Graph also offers a hierarchical grouping using a tree representation on Y-Axis where parent nodes can be expanded/collapsed to enable hiding/showing only a subset of leaf channels. User can easily configure how to form the hierarchy by choosing a separator for this purpose. The default separator is '.' for breaking down the key value strings into a hierarchy. Things get plotted inside the graph same way as before but only for the leaf nodes of the tree. Switching between flat mode and tree mode is achieved using the right most action button on the toolbar.


Sorting on Y-axis

Using the small triangle on the top left corner of the graph, y-axis channel names can be sorted in an alphabetical order. Clicking on the triangle, sorting is switched from un-sorted to ascending and descending sorts. Sorting feature also works in tree mode where everything up to leaf nodes get sorted.


Other features

  • Y-Axis separator can be dragged to make more room for data plotting. Mouse over channel names show the complete tooltip.
  • Mouse-Wheel scroll anywhere in the bottom x-axis area, triggers zooming while anywhere else causes vertical scroll.
  • Dragging of mouse in a x-axis region, zooms to the selected region. Alt+drag, zooms inside the graph
  • View also offers a region at the bottom that can be populated with status and progress information
  • State transitions can be shown as a longer than normal black separators, this can be optionally on or off
  • Error conditions for state presentation type can be configured to be displayed as a dashed line, instead of a bar
  • Measurement marker values and deltas will be displayed in a reserved region at the top which again is draggable when more room is needed.


Technical Notes / Advanced

To present Y-Axis, DVT Graph can be configured to autodiscover in three dimensions based on data in the connected buffer:

  • Channel Names
  • States/Events in the Channel
  • Channel Presentation types

Tooltips are supported on mouse over on states/events by default but user may also add any additional tooltips they'd like to be displayed along with the defaults as well.
Annotation can also be added to the states/events drawn on DVT Graph globally or per channel.
Aliases can be used to present Y-Axis keys differently or move certain channels to subtrees using '.' notation

To change the dvt graph display you can Right-click on the graph and select Display Properties
