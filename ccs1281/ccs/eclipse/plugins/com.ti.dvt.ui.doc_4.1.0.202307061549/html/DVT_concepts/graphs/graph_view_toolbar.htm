Graph view toolbar

Note: Some graphs may have a subset of the following buttons or some may not be applicable and hence disabled.

Each graph view has a toolbar with several options to control the entire graph:

  • The (Enable Grouping) button on any graph associates or disassociates this graph within a specified group. See the Groups Feature topic.

  • The (Align Horizontal Center in Group) button forces all the grouped views to use the same horizontal value reference center as the current view.

  • The (Align Horizontal Range in Group) button forces all the grouped views to use the same horizontal range (left and right X-axis values) as the current view.

  • The (Measurement Marker) icon triggers adding a measurement marker line to a view. See the Measurement Marker Feature topic.

  • The (Zoom In) button increased the graph resolution to provide more detail by the set zoom factor and zoom directions. Alternatively, you can press the ALT key and drag the mouse on any graph area of the graph.

  • Use the (Zoom Out) button to decrease graph resolution and see more data using the set zoom factor and zoom directions.

Note: The Zoom Options help you control the display by specifying what combination of graph axes and resolution factors to use.

  • Use the (Zoom Options) button to reset the zoom for the graph (return to original zoom factor and orientation). Use the drop-down selection arrow next to the button to specify the zoom factor and X-Y directions. Some graphs can only adjust zoom on the X-axis.

  • The (Refresh) button updates the view when pressed.

  • The (Find) button displays the Find dialog to search for a specific event or value. See the Find Feature topic.

  • Use (Enable Filter) button to display the Set Filter Expression in <view title> dialog. See the Filter Feature topic.

  • Use the (Freeze data update) button to halt the data refresh for this graph. Clicking the button again resumes the data refresh period.

  • (Menu) icon does not contribute any functionality for graphs.

  • (Minimize) icon reduces the view to just the dialog tabs and graph view toolbar.

  • (Maximize) icon enlarges the view to the entire available window size.

  • (Restore) icon returns a view to its original size, prior to using the Minimize or Maximize icons.
