Graph view display controls

Graph views have multiple types and therefore display slightly different toolbars, shortcut menus, and Display properties for each different type:

  • Graph view controls

  • Graph view types

  • Mouse-over interactions

Graph view controls

The Features use three sets of controls to provide the graph display functionality:

  • Toolbar—each graph view displays this toolbar as part of the view. See Graph view toolbar topic.

  • Shortcut menu - right-clicking in any graph displays the shortcut commands . Each graph type has unique functionality, however see the Common graph shortcut commands topic.

  • Display Properties - Normally these properties are not changed for graphs unless you want to customize the look-and-feel of the displays. See Display Properties topic for additional details.

Graph view types

Graph views can be characterized as:

  • Line Graph

  • State Graph

  • Discrete Graph

  • Bar Graph

Mouse-over actions

Click and hold the left mouse button within any graph to see a cursor which reveals one or more axis values. Release the left mouse button to stop displaying values

You can click and hold the left mouse button to continuously move the cursor. As the cursor moves in your primary graph view, all tables and graphs that are grouped with this view, change to reflect the cursor position.
