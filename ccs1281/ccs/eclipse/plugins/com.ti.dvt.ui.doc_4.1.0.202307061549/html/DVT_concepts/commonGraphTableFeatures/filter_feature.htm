Filtering displayed data

Use the (Enable Filter) drop-down menu or Ctrl+T shortcut to display the Set Filter Expression dialog and display only information that meets your specified criteria.

With Set Filter Expression in <view title> dialog, you can restrict the data to apply to only specific functions, drivers, time windows, and so forth.

When you want the table or graph to display its original, complete set of data, click the () Disable Filter toolbar icon.

The Set Filter Expression in <view title> dialog can select data by specifying a criteria using two methods:

  • Individual field names (plus value and an operator)

  • Regular expression

 

Note: Bit mask values must be in hexadecimal format.

Note: For help with regular expression syntax, place the cursor in the Use Expression field and press the Ctrl + Space keys to display the expression assistance drop-down list

Click Clear History to clear the list of previously used filter expressions.
