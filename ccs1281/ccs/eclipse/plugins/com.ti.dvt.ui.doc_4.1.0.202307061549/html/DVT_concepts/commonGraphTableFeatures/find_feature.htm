Finding logged data information

Use the Find in <view title> dialog to locate a value containing a particular string in one of the fields, or a value whose fields satisfy a particular expression. Insert a field and a value (or expression) and click Find. The view shifts to the first occurrence of the data value. Clicking Find repeatedly, moves you through the data to each location of the matched string.

Click the (Find) button to display the Find in <view title> window. The <view title> depends upon which table or graph view is displaying the Find toolbar icon.

With the Find <view title> dialog you can:

  • Search for a specific value in a specific table/graph field

  • Use a regular expression to search for a string or value

  •  Enable Use Bits Mask: (long) and specify a bit mask in the adjacent field to exclude a portion of a value from consideration.

  • [Optional] Configure the features in the Options area to refine the data search:

    • Select Use Expression to search with a regular expression entered in the Find field only

    • Select Wrap Search to search throughout the entire data range, regardless of your current location

    • Select Exact Match to exclude nearest matches using the "like" operator or differences in letter case from matching

    • Select Case Sensitive to force the match to include case-sensitivity in its comparisons

To clear the list of previously searched items in the Find field, click Clear History.
