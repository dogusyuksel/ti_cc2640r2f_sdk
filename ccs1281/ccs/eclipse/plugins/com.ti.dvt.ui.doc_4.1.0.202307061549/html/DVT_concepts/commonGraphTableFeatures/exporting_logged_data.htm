Exporting data to other applications

You can export any data in a table or graph view to a comma-separated value CSV file (*.csv filename extension) to perform more extensive processing or create more elaborate displays. All columns (displayed or hidden) for a table or graph are exported. Microsoft Excel and other applications can read a CSV file as demonstrated in these graphics.

To export the data in any table or graph view:

  1. Right-click in any DVT Solution table or graph view, select the Data ® Export shortcut command.

  2. When the Save As window displays, navigate to the directory where you want to store this information.

  3. Type a name for this data in the File name field and click OK.
