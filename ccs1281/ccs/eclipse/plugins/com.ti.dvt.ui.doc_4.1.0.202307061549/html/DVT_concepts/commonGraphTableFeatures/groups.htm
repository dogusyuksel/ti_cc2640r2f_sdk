Grouped tables and graphs

Graphs within a DVT Solution can be linked together as a group, allowing exact points of data to be displayed on different graphs simultaneously. Group selection helps you to highlight a specific item in the views. Once the graph has been added to a group and activated, you can perform various functions on all the views in the group at once, including horizontal scrolling, group selection (or X cursor), horizontal center alignment, and horizontal range alignment. Several toolbar icons help you take advantage of group features.

Use the (Groups) button to enable or disable the view from responding to changes from other views in the group.

Use the (Align Horizontal Center in Group) button to cause all the views in the group to use the same horizontal center as the current view.

Use the (Align Horizontal Range in Group) button to cause all the views in the group to use the same horizontal range (left and right values) as the current view.

Note: Group alignment options are disabled for graphs not added to a group.

With the group commands found in the graph shortcut menus, you can:

  • Create a new named group to associate with different views

  • Add a graph to an existing group

  • See the list of groups created

  • Remove a named group

See Managing groups and Synchronizing tables and graphs for specific procedures.
