Zooming

Zooming is only available on graphs. You can zoom in both X-Axis and Y-Axis in Line Graph while for DVT Graph, you may only use the zoom on the X-Axis.


Zoom feature can be used by any of these methods:

Using Mouse

  • Use ALT_Mouse Drag to select any area on the graph to Zoom-In
  • Also Mouse Drag by itself on the axis areas (without holding the ALT key) would provide Zoom-In on the selected area
  • You may also use Scroll-Wheel on the X-axis areas for Zooming-In as well as Zooming-Out

Using Keyboard

  • CTRL_+ to Zoom-In
  • CTRL_- to Zoom-Out

Using View Toolbar

  • : The Zoom In button on toolbar increases the graph resolution to provide more detail by the set zoom factor and zoom directions.
  • : The Zoom Out button to decrease graph resolution and see more data using the set zoom factor and zoom directions.
  • : Use this to reset the zoom for the graph (return to original zoom factor and orientation). Use the drop-down selection arrow next to the button to
  • : With the dropdown right next to Reset Zoom, known as zoom options dropdown, you can specify the zoom factor and directions of the zoom for line graph:
    • : Horizontal zoom
    • : Vertical zoom

Note: When keyboard, scroll-wheel or toolbar buttons are used for zooming, the cursor position is used as a point of reference to apply zoom, and when no cursor position is found, the center of the graph is used.

To set the cursor position, simply click on the point of interest on the graph area, this will leave behind a red line (or cross-hair) which will get used for zooming.
