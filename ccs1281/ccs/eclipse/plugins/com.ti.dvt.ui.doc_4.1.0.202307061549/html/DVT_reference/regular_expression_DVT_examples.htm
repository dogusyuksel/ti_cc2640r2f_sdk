Examples of regular expressions

Suppose you wanted to find records having data in field Source begin with the word Audio

  • Data Field—Source

  • Regular Expression—Source like "Audio.*"

To find a Trace Analyzer Function Name that contains the letters __divu

  • Data Field—Function

  • Regular Expression—Function like ".*__divu.*"

Matching a floating point number

  • [-+]?[0-9]*\.?[0-9]+([eE][-+]?[0-9]+)?
