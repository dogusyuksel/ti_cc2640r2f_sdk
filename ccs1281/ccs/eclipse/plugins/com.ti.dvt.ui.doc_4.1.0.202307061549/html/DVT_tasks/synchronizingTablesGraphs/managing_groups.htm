Managing groups

Every instance of data view lets you to define one or more groups that help you correlate different graph information plotted along a common reference. Often this reference is elapsed time or cycles or sequence number.

Creating a new group

  1. Right-click anywhere inside a graph.

  2. Select Groups -> New group shortcut command or press Ctrl+G.

  1. In the Create group dialog box, specify a group name (such as time) and click OK.

Adding a view (table or graph) to an existing group

  1. Right-click anywhere on the graph;

  2. Select Groups and then select the name of the group from the cascading group list

Note: If the solution does not contain any groups, this option is disabled.

Deleting a group from the group list

  1. Right-click anywhere inside a graph.

  2. Select Groups -> Delete group shortcut command or press Ctrl+Alt+G.

  3. In the View Groups box, select the group name in the list and .

  1. Click Delete, then click Close.

To remove a view permanently from the group

  1. Select the view you want to remove.

  2. Right-click the mouse and select the Groups ® <none> shortcut command.

Even if you click the Activate View with Group toolbar or Enable Grouping shortcut command, this view does not respond to actions performed by the group.

To temporarily disable a view from responding to group actions

  1. Select the view you want to disable.

  2. Click the Deactivate View with Group toolbar icon.

Note: Click the Activate View with Group toolbar icon and the view responds to the group actions.
