Advanced Tools Preferences

These preferences only apply to some of the features listed under Tools menu like RTA, Trace, Profiler, ...

Temp Directory

When dealing with large data sets(like in RTA, Trace, Profiler,...) and to minimize the memory usage, file buffers are used. By using file buffers, only the data that is visible to the users get loaded in the memory. These files are temporary and as soon as user closes Code Composer Studio, they get deleted.

These files could grow fast based on how long you are collecting data for, and what data you are collection. It is recommended to change this setting to a different drive than your system drive.


Error Log View

To report errors, DVT features use eclipse' standard Error Log View. You can specify based on the severity of error, the Error Log view, to get openned, or get the focus.

Note: To display the Error Log at any time, choose the Window®Show View®Others command and select the Error Log found in the PDE Runtime folder of the Show View window.
