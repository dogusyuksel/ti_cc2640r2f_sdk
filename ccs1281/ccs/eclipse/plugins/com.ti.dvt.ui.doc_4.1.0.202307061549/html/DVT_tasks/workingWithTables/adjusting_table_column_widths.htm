Adjusting table column widths

You can resize the widths of table view columns by:

  • Using the Auto Fit Columns button to adjust every column width to display all visible data.

  • Individually, selecting a column and manually resizing its width

To display all the visible information in a table view, click the (Auto Fit Columns) button on the table view toolbar. The Auto Fit Columns examines all visible records for each column to ensure that no data is displayed in a truncated format.

After Auto Fit Columns command

To manually resize a table view column:

  1. Place your mouse cursor on the right column division marker.

  1. When your cursor changes to a resize cursor, click the mouse button (MB1) and hold down.

  2. Drag the mouse so all the data for this table view column displays.

  3. Repeat steps one through three to resize all other desired columns.
