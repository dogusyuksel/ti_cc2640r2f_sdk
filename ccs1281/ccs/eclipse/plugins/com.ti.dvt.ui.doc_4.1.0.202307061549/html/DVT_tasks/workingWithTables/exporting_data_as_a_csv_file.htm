Exporting data as a CSV file

You can save the data in a table or graph view to an external file by using the Data ® Export command. All rows and columns contained in the table (not just the displayed columns) and the displayed graph numbers are placed into a comma-separated value file format (*.csv filename extension). The numbers display in the CSV format using a general format. You can use software such as Excel to perform additional computations or create annotated charts from the exported information.

To export the data in an external CSV file:

  1. Select a table or a graph view.

  2. Right-click and select the Data ® Export command from the shortcut menu.

  3. When the Save As window displays, use the icons to the right of the Save in field to navigate to the desired directory folder.

  4. Select an existing file name or type a file name into the File name field.

  1. Click OK. The software saves all view information in the named file.

  2. Open the file using a spreadsheet software program such as Excel, to begin working with the information.
