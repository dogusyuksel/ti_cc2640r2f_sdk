Usage: E:\Work\tools\TI\ARM\arm_4.1.7\bin\hex470 [-options | -h] filename

Options:
 -@<file>      Read commands from 'file'
 -q            Quiet Operation
 -e<addr>      Specify entrypoint address or symbol name
 -o<file>      Specify output file names
 -image        Select image mode
 -linkerfill   Include linker fill sections in images
 -zero         Zero based addressing
 -fill<x>      Specify fill value
 -romwidth<x>  Specify rom width
 -memwidth<x>  Specify memory width
 -order        Specify data ordering (endianness)
 -bootorg<adr> Specify origin address or symbol of boot table ROM
 -map<file>    Specify map file name

Output Format Specification:
 -a            Output ASCII hex format
 -i            Output Intel hex format
 -m[1|2|3]     Output Motorola S hex format
 -t            Output TI-Tagged hex format
 -x            Output Extended Tektronix hex format
 --ti_txt      Output TI-TXT hex format

            See the User's guide for additional information
            and target specific options.
