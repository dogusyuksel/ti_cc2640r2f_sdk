EnergyTrace™ Technology

The EnergyTrace™ technology is an energy-based code analysis tool set that is useful for measuring and viewing the application’s energy profile and optimizing it for ultra-low power consumption. MSP430 devices with built-in EnergyTrace™ Technology support allow real-time monitoring of many internal device states while normally executing user program code.

The EnergyTrace™ Technology is supported on selected MSP430 devices and debuggers.

Please refer to the online help or to the CCS for MSP430 User‘s Guide (SLAU157) for details.
