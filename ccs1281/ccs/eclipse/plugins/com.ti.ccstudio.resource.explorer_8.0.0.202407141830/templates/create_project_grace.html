To set up a project based on an existing example:

  1. Click on the link above to copy the files to the desired destination directory.
  2. In the Grace tool, select File->Open and navigate to the directory in which you copied the files.
  3. Select the .cfg file to open.
  4. In the Configuration dialog box, specify the target device.
  5. At this point you can proceed to use the Grace tool to configure the device’s peripherals.
  6. Click on the Generate toolbar button to generate the C files that will initialize your peripherals.
