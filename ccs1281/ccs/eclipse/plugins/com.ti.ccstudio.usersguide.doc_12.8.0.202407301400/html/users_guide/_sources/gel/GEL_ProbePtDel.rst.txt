GEL_ProbePtDel()
----------------

Deletes a breakpoint.

**Syntax**

**GEL_ProbePtDel(**\ address\ **);**

**Parameters**

addressspecifies the address where a breakpoint will be deleted.

**Description**

This function allows you to delete a breakpoint at a specified address,
if the breakpoint exists.

**Synchronous**

Synchronous from GEL: Yes

Completely synchronous: No

**Example**

GEL_ProbePtDel(100);
