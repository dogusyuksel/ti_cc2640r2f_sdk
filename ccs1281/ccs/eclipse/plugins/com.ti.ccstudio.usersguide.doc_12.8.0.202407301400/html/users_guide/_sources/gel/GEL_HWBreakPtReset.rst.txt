GEL_HWBreakPtReset()
--------------------

Removes all hardware breakpoints.

**Syntax**

**GEL_HWBreakPtReset();**

**Parameters**

None.

**Description**

This function removes all hardware breakpoints currently set, whether
they are enabled or disabled.

**Synchronous**

Synchronous from GEL: Yes

Completely synchronous: No
