GEL_MemoryLoadData()
--------------------

Loads a block of memory from a file.

**Syntax**

**GEL_MemoryLoadData(**\ startAddress\ **,** page\ **,** length\ **,
"**\ fileName\ **" )**

**Parameters**

startAddress is the first address in the block.

page identifies the type of memory to fill: 0 (Program memory), 1 (Data
memory) or 2 (I/O space)

For processors that do not have more than one type of memory, use 0 for
this Parameters. For simulated targets, the I/O space Parameters is not
supported.

length defines the number of values to load from the file

fileName name of the file to load data from. The fileName must be
enclosed in quotation marks.

**Description**

GEL_MemoryLoadData() is an extention to GEL_MemoryLoad() and supports
loading data in addtional formats to target memory from a specified
file.

**Synchronous**

Synchronous from GEL: Yes

Completely synchronous: Yes

**Example**

GEL_MemoryLoadData( 0x100, 0 , 0x100, "c:\\mydir\\myfile.dat" );
