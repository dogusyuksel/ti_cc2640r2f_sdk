GEL_Trace()
-----------

Enables GEL execution trace

**Syntax**

**GEL_Trace(** level **);**

**Parameters**

level is the trace level: 0 - no trace, 1 - trace entry and exit of
functions, 2 - trace execution of each statement.

**Description**

Sets the trace level of the GEL execution trace.

**Synchronous**

Synchronous from GEL: Yes

Completely synchronous: Yes

**Example**

| GEL_Trace(0);
| GEL_Trace(1);
