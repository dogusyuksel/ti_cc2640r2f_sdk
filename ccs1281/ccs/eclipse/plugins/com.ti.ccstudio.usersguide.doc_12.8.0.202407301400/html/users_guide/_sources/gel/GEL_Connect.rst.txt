GEL_Connect
-----------

GEL built-in function that can be called on a disconnected target to
connect it.

**Syntax**

**GEL_Connect()**

**Parameters**

None

**Description**

The function connects the target and then returns. It throws an
exception if the target is already connected or fails to connect.

**Synchronous**

Synchronous from GEL: Yes

Completely synchronous: Yes
