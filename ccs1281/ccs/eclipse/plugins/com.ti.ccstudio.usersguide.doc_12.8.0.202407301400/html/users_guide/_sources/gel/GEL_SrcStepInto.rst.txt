GEL_SrcStepInto()
-----------------

Steps into source code.

**Syntax**

**GEL_SrcStepInto()**

**Parameters**

count(optional) specifies the number of times the command is to be
repeated. If no count is specified, executes a single assembly
instruction.

**Description**

Executes the next available instruction in C or assembly source code
then halts. If an action enters an area with no source code, the
debugger continues stepping until source code (somewhere) is reached.
Repeats count times.

**Synchronous**

Synchronous from GEL: No

Completely synchronous: No
