GEL_MapReset()
--------------

Resets the memory map.

**Syntax**

**GEL_MapReset()**

**Parameters**

None.

**Description**

This function resets the memory map by making all memory non-readable
and non-writeable.

**Synchronous**

Synchronous from GEL: Yes

Completely synchronous: Yes

**Example**

GEL_MapReset();
