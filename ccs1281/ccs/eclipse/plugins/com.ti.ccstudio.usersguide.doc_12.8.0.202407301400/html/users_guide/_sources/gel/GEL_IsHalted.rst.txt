GEL_IsHalted
------------

GEL built-in function that can be called at any time to determine if the
target is halted.

**Syntax**

**GEL_IsHalted()**

**Parameters**

None

**Description**

The function returns 1 if the target is halted and returns 0 otherwise.

**Synchronous**

Synchronous from GEL: Yes

Completely synchronous: Yes
