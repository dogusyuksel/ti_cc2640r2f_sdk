GEL_DisableFileOutput()
-----------------------

Disables writing GEL Text output to a file

**Syntax**

**GEL_DisableFileOutput();**

**Parameters**

none

**Description**

Disables writting GEL Text output to file

**Synchronous**

Synchronous from GEL: Yes

Completely synchronous: Yes

**Example**

GEL_DisableFileOutput();
