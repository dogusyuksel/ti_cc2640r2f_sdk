GEL_BreakPtDisable
------------------

Disables a software breakpoint at the specified address.

**Format**

**GEL_BreakPtDisable(**\ address\ **);**

**Parameters**

address specifies the location of the software breakpoint.

**Description**

The software breakpoint at the specified address is disabled. If the
breakpoint does not exist or is already disabled, no action is taken.
This function will not disable the breakpoint if it is a hardware
breakpoint.

**Synchronous**

Synchronous from GEL: Yes

Completely synchronous: No

**Example**

GEL_BreakPtDisable(0x2000);

GEL_BreakPtDisable(main + 100);
