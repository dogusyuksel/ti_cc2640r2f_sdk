GEL_ProbePtAdd()
----------------

Adds a software breakpoint at the specified address.

**Syntax**

**GEL_ProbePtAdd(**\ address\ **);**

**Parameters**

addressspecifies the location of the breakpoint. If address is a string
Parameters, a symbolic breakpoint will be set at that location and
re-evaluated whenever symbols change. Otherwise, a breakpoint will be
set at the specified address.

**Description**

This function sets a software breakpoint at a specified address. The
address can be any GEL expression that evaluates to a program address,
or a string that specifies a symbolic location. The breakpoint must be
manually connected to a window or a file I/O operation before it will do
anything.

**Synchronous**

Synchronous from GEL: Yes

Completely synchronous: No

**Example**

GEL_ProbePtAdd(PC + 100);

GEL_ProbePtAdd(0x2000);

GEL_ProbePtAdd("main");
