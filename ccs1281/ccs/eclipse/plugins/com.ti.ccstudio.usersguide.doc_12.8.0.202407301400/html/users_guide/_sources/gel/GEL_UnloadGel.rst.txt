GEL_UnloadGEL()
---------------

Unloads a GEL file.

**Syntax**

**GEL_UnloadGel("**\ fileName\ **");**

**Parameters**

fileNamenames the GEL file to be unloaded. The fileName must be enclosed
in quotation marks.

**Description**

This function unloads the functions defined in the specified GEL file.

The fileName Parameters is case sensitive. The filename must be
specified exactly as it was when the file was loaded. If the file is not
in the current directory, the fileName Parameters must provide the full
path. A double backslash escape sequence is required to ensure that you
get a backslash into the fileName Parameters.

**Note**: When a file is opened using menu commands, Windows capitalizes
the drive letter.

**Synchronous**

Synchronous from GEL: Yes

Completely synchronous: Yes

**Example**

GEL_UnloadGel("C:\\MyDir\\myfile.gel");
