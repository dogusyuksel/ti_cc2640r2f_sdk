GEL_ProbePtReset()
------------------

Deletes all breakpoints.

**Syntax**

**GEL_ProbePtReset();**

**Parameters**

None.

**Description**

This function can be used to remove all currently set breakpoints.

**Synchronous**

Synchronous from GEL: Yes

Completely synchronous: No

**Example**

GEL_ProbePtReset();
