GEL_BreakPtReset()
------------------

Clears all breakpoints.

**Syntax**

**GEL_BreakPtReset();**

**Parameters**

None.

**Description**

This function clears all software breakpoints.

**Synchronous**

Synchronous from GEL: Yes

Completely synchronous: No

**Example**

GEL_BreakPtReset();
