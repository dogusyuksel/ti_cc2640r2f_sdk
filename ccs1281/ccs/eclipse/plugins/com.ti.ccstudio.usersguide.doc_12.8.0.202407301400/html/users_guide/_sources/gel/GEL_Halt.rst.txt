GEL_Halt()
----------

Stops execution.

**Syntax**

**GEL_Halt();**

**Parameters**

None.

**Description**

This function halts the target program if it is executing.

**Synchronous**

Synchronous from GEL: No

Completely synchronous: No

**Example**

GEL_Halt();
