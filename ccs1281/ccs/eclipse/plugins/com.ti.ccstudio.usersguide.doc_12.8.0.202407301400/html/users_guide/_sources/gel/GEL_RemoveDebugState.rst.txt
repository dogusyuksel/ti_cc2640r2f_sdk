GEL_RemoveDebugState
--------------------

Removes the debug state from the target.

**Format**

**GEL_RemoveDebugState();**

**Parameters**

None

**Description**

This function removes the debug state from the target, much like what
happens when the target is disconnected. The debug state includes
hardware and software breakpoints, global breakpoints, profiling state,
and anything else that might effect target execution. The state can be
restored using GEL_RestoreDebugState or selecting Restore Debug State
from the Debug menu.

**Synchronous**

Synchronous from GEL: No

Completely synchronous: No

**Example**

GEL_RemoveDebugState();
