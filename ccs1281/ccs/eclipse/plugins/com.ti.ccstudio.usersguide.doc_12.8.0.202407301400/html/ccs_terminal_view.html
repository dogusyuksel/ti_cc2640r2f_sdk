<!-- Start of markdown source --> Many embedded applications include a serial connection for displaying messages, sending commands or checking status. In order to interact with such a connection a terminal utility is required. The **Terminal View** in Code Composer Studio can be used for this. ## Video ## <iframe width="854" height="480" src="https://www.youtube.com/embed/7-CxZuAn-Hc" frameborder="0" allowfullscreen></iframe> ## Opening ## To open this tool go to the **View** menu and select **Terminal**. ![](./images/terminal_view_menu.png) This will open the view in your workspace. ![](./images/terminal_view.png) ## Connecting ## Click on the **Open Terminal** button on the toolbar. ![](./images/terminal_view_open_button.png) This will open a dialog box where you can configure your connection. ![](./images/terminal_view_launch_dialog.png) Typically you will configure the terminal for a serial connection. ![](./images/terminal_view_launch_dialog_serial_connection.png) This is an image of a terminal connected to the portable demo that comes with the CC13x0 SimpleLink SDK. ![](./images/terminal_view_connected.png) ## Multiple Connections ## It is possible to open multiple terminals by clicking on the ![](./images/terminal_view_new_button.png) This will open another terminal view and bring up the settings dialog box. <!-- End of markdown source --> <div id="footer"></div>
