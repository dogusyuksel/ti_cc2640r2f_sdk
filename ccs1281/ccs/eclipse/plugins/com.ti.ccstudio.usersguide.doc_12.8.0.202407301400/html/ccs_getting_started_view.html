#Getting Started View# The Getting Started View is the first screen that is visible once you have launched Code Composer Studio and selected a workspace. ![](./images/ccsv8_getting_started_view.png) ##Getting Started Video## In the middle of the view is a link to the Getting Started Video. This video provides a quick overview of Code Composer Studio. ##Actions## The purpose of the Getting Started View is to provide fast access to common actions and tasks, especially those that are relevant when first starting to use Code Composer Studio. ###Resource Explorer### This will launch Resource Explorer and allow you to browse through available example projects, demo applications, datasheets, and more for your chosen platform. ###New Project### Starts the Project Wizard and enables you to create a new empty project for your selected device. ###Import Project### Select this option if you have an existing Code Composer Studio project that you wish to import into your workspace. ###App Center### The App Center is used to install additional development tool capabilities. ###Support Forum### The E2E support forums are the place to go to get technical support. ###Videos### There is a wide selection of videos available that explain how to accomplish various tasks in Code Composer Studio. ##Simple Mode## Code Composer Studio includes many features and capabilities. If you are more comfortable with a more simplified environment you may want to give Simple mode a try. Simple mode strips down the user interface to just the most basic functionality. Once you are comfortable with the environment you can use the Getting Started view to disable Simple mode. <!-- End of markdown source --> <div id="footer"></div>
