Over the Air Download (OAD)¶

This section serves as a guide to the Texas Instruments Over-the-Air Download (OAD) ecosystem. OAD is a device firmware upgrade method that allows the firmware image running on a device to be updated over the air using a proprietary RF connection while providing power loss protection.

The OAD guide section will cover:

  • OAD theory of operation and architecture

  • How to run the OAD examples included in the SimpleLink CC2640R2 SDK

  • How to add OAD to an existing sample application

The supported development kit for OAD is a launchpad. To follow the procedures listed in this guide, two launchpad development kits are required.
