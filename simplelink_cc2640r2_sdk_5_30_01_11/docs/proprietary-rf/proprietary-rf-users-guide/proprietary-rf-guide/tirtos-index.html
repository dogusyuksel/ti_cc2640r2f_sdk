TI-RTOS (RTOS Kernel) Overview¶

TI-RTOS is the operating environment for EasyLink projects on CC2640R2 devices. The TI-RTOS kernel is a tailored version of the legacy SYS/BIOS kernel and operates as a real-time, preemptive, multi-threaded operating system with drivers, tools for synchronization and scheduling.
